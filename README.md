# Mizan

โปรแกรมตัวอย่างสำหรับการเก็บข้อมูลผู้ติดต่ออย่างง่าย เขียนด้วยภาษา Python
สามารถรันผ่านเทอร์มินัลเพื่อกรอกข้อมูลและบันทึกเป็นไฟล์ JSON ได้ทันที

## วิธีใช้งาน

1. ติดตั้ง Python 3.10 ขึ้นไป
2. รันคำสั่งต่อไปนี้ในไดเรกทอรีโปรเจกต์

```bash
python collector.py
```

โปรแกรมจะถามชื่อ อายุ อีเมล เบอร์โทรศัพท์ และหมายเหตุ (เว้นว่างได้)
เมื่อกรอกข้อมูลเสร็จจะถูกบันทึกไว้ในไฟล์ `collected_data.json` สามารถเปิดดูหรือ
แก้ไขด้วยโปรแกรมแก้ไขข้อความทั่วไปได้เลย

## โครงสร้างโค้ด
- `collector.py` มีคลาส `DataCollector` สำหรับอ่าน/เขียนข้อมูลจากไฟล์ JSON
- ฟังก์ชัน `run_cli` ใช้แสดงคำถามและรับอินพุตจากผู้ใช้ทีละรายการ
- มีการบันทึกเวลา (`timestamp`) ของแต่ละข้อมูลเพื่อใช้อ้างอิงภายหลัง
